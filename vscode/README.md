# openPly for VS Code

Free, local-first AI coding assistant. Your code stays yours.

## Commands

- **openPly: Start in Terminal** — Opens the openPly CLI in a VS Code terminal
- **openPly: Chat** — Ask openPly to do something
- **openPly: Explain Selected Code** — Explain the selected code

## Install

```bash
npm install -g openply
```

Then use the commands above.
